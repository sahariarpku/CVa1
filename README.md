# CVAI: Academia Job & CV Manager

A comprehensive application for managing academic job applications and tailoring CVs using AI.

## Features
- **Job Discovery:** Swipe-based job discovery interface.
- **Application Tracker:** Kanban-style tracking for your applications.
- **CV Builder:** AI-powered CV builder and editor.
- **AI Tailoring:** Automatically tailor CVs and cover letters for specific job descriptions.
- **Export:** Download resumes in PDF and Word formats.

## Tech Stack
- **Frontend:** React, Vite, TailwindCSS
- **Backend:** Node.js, Express
- **Database:** Firebase (Firestore, Storage, Auth), Supabase (optional/legacy)
- **AI:** Google Gemini

## Getting Started

1.  Clone the repository.
2.  Run `npm install` in the root directory.
3.  Run `npm run dev` to start both client and server.
4.  Open `http://localhost:3001` (or 3000).
